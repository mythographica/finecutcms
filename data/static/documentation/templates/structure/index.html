Each template consists of three files:

— Template itself (index.html).

— Page Settings Snippet (header.txt).

— Page Properties Snippet (snippet.txt).


Templates use the FineCut template engine with {{property}} syntax for variable substitution and {{#if condition}} for conditionals. Components are included via {{>helperName}}.
